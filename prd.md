# Kalinova Product Requirements Document (PRD)

## 1. Product Vision

Kalinova simplifies Kali Linux by providing a graphical, beginner-friendly security suite that abstracts CLI complexity and intelligently guides users.

---

## 2. Target Users

Primary:
- Cybersecurity students
- Beginners in ethical hacking

Secondary:
- Educators and trainers

---

## 3. Problem Statement

Kali Linux:
- Contains 600+ tools
- Is CLI-heavy
- Lacks guided workflows

Kalinova:
- Provides GUI wrappers
- Reduces cognitive load
- Suggests next actions using ML
- Structures output intelligently

---

## 4. Core Features

- Clickable GUI tools
- Terminal-free execution
- Tool-type adaptive output
- ML next-step suggestion
- Risk classification
- Beginner explanation panel
- Selective reporting
- .deb installation

---

## 5. Success Metrics

- No CLI required
- Stable execution
- ML accuracy > 70%
- Clean installation & removal
- Positive beginner usability feedback

---

## 6. Risks

- Poor ML dataset quality
- Tool output format changes
- Overengineering UI

Mitigation:
- Controlled scope (6–8 tools)
- Modular architecture
- Incremental testing
